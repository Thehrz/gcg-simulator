## Build

### 构建前须知

本仓库不包含原神游戏资源, 故用户启动本仓库代码后无法在浏览器内查看到完整的游戏资源, 包括卡牌, 头像, 技能等图片资源。

**您需要在不违反 [米哈游原神游戏使用许可及服务协议](https://ys.mihoyo.com/main/company/agreement) 的前提下 (不通过解包等行为) 自行获取原神游戏资源以正常启动本项目。**

### 构建命令

```
yarn build
```
